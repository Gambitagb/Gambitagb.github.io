Tipos de selectores que entran en examen

Clase (class=””)

El selector selecciona elementos con un atributo de clase específico..class Para seleccionar elementos con una clase específica, escriba un carácter de punto (.), seguido del nombre de la clase. También puede especificar que sólo elementos HTML específicos se vean afectados por una clase. Para ello, comience con el nombre del elemento, luego escriba el carácter punto (.), seguido del nombre de la clase.

Identificador (id=””)

El selector aplica estilos al elemento con el identificador especificado

Todos los elementos (*)

El selector selecciona todos los elementos.*
El selector * también puede seleccionar todos los elementos dentro de otro elemento

Elementos HTML también puede referirse a más de una clase

Varios elementos ( p, em, h1, h2)

El selector de elementos selecciona todos los elementos con el nombre de elemento especificado.

Elementos contenidos en otros elementos (ol li)

Sobrevuleo (p:hover)

El selector se utiliza para seleccionar elementos al pasar el ratón sobre ellos.:hover Propina: El selector :hover se puede utilizar en todos los elementos, no sólo en los enlaces.
Propina: Utilice el selector :link para aplicar estilo a los vínculos a páginas no visitadas, el selector :visited para dar estilo enlaces a páginas visitadas y el selector :active para aplicar estilo al vínculo activo.
Nota: :hover DEBE venir después de :link y :visited (si están presentes) en la definición de CSS, para que sea efectivo.
